# CampusPilot

## תיאור הפרויקט
CampusPilot היא מערכת לניהול קמפוס אקדמי, המיועדת לסטודנטים, אנשי סגל ומנהלי מערכת.  
המערכת מאפשרת ניהול משתמשים, קורסים וסטודנטים בצורה נוחה וברורה דרך ממשק אינטרנטי.

## מטרות הפרויקט
- ניהול משתמשים (הרשמה והתחברות)
- הפרדה בין תפקידי משתמשים (Admin / Staff / Student)
- ניהול סטודנטים וקורסים
- עבודה עם בסיס נתונים MySQL
- שילוב צד שרת (PHP) וצד לקוח (HTML, CSS, JavaScript)

## טכנולוגיות
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- GitHub

## מבנה הפרויקט
- assets/ – קבצי עיצוב ותמונות
- includes/ – קבצי עזר וחיבור לבסיס נתונים
- phpLogic/ – לוגיקת צד שרת
- sql/ – קבצי SQL
- login.php / register.php – התחברות והרשמה
- index.html – דף ראשי
